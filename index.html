Feature Selection Based Transfer Subspace

发表于 2018-05-14

摘要

跨语料库的语音情感识别最近很受关注，通常包括两个基本问题—特征匹配和特征选择。作者提出了新颖的算法来处理这两个问题（之前的研究都是分别地研究两个问题或只研究其中一个问题）。为了解决第一个问题，通过减少不同语料库的差异和保留重要属性来学习潜在的公共子空间。同时，在投影矩阵上使用$l_{2,1}-norm$来解决第二个问题。为了保证子空间的鲁棒性和具有辨识力，探索了数据的几何信息。

介绍

GMM、NN、SVM、supervised dic-tionary learning, multi-task learning, sparse representation、DNN这些方法的效果某种程度上说还不错，但是是建立在训练集和测试集在同一个语料库中的。

在语音识别中，模型的迁移性十分重要。研究者用cepstral mean normalization (CMN), maximum a pos-teriori (MAP), joint factor analysis (JFA), vocal tract length normalization (VTLN), maximum likelihood linear regres- sion (MLLR)等方法来促进模型的性能。同时，随着深度学习的发展，发现DNN可以从未经处理的数据学习到鲁棒的特点，有很多研究CNN，LSTM的成果。共同的策略是学习高层次的不变情绪特征。

上述的方法都没有考虑到语料库偏差的影响，迁移学习最近被提出来解决这一问题。它是通过从源数据学习到的知识应用到目标数据来实现的。随后列举了一些算法，所有这些算法都着眼于寻找潜在的共同特征表示来处理特征匹配问题，而不考虑特征选择的重要性。

子空间学习是把高维的数据降维到低维的特征空间，同时重要的统计特征被保存。随后列举了很多算法，作者把它们都归为general graph embedding framework。但是子空间学习假设训练集和测试集独立同分布，对于跨语料库的情感识别效果不好。为了解决它提出了基于迁移学习的子空间学习。但这类方法不是最优的，因为既不考虑特征选择的重要性，也不考虑数据的几何结构，这两者对于情感分类都很重要。

这篇文章提出了feature selection based transfer subspace learning，方法的核心思想是转移子空间学习和特征选择一起进行。另外，还用到了正则化来提高模型的表现（鲁棒性和判别能力）

本文模型的总结：

  • FSTSL将转移子空间学习和特征选择优雅地结合到一个统一的框架中.
  • 在学习投影矩阵时，提出了一个图正则化项来显式地保存数据的几何结构.
  • 提出了迭代算法求解联合学习目标函数,表现优于最先进的迁移学习算法.

相关工作

特征选择

特征选分为监督、非监督和半监督。非监督学习很难，但是，大量未标记的数据可以使特征选择变得现实。大部分的传统特征选择方法不重视特征之间的相关性。现在的研究y已经证明$l_{2,1}$范数对稀疏诱导特征选择非常有用。作者提出的算法在转移学习下同时执行特征选择和子空间学习。

迁移子空间学习

先介绍了当前的一些研究，这些研究都是分别地研究两个问题或忽略特征选择问题（子空间学习和特征选择）。情感特征的维数通常都是很高的，自然地会存在冗余或无关的特征，所以对于跨语料库的语音情感识别如何同时选择特征很重要。作者的方法将转移子空间学习，$l_{2,1}$范数和图正则化结合为一个通用框架。

实验

Data sets

用了EMO-DB,FAU Aibo和eNTERFACE三个语料库。

实验设置

作者使用openSMILE工具包来提取声学特征，使用计算语言学挑战INTERSPEECH 2013的标准特征集来评估试验结果。设计了六种类型的实验来评估跨语料库语音情感识别，即上述数据集中两两结合分别作为训练集和测试集，每个实验训练集有标签，测试集没有标签。在每个实验中，每个语料库被分成五个部分，其中源语料和目标语料的随机4/5被用于训练，而其他语言被选择用于评估，为了覆盖训练集和测试集的可能情况，实验重复了10次试验。

作为比较的算法

作者将实验结果与一个基线和几个最先进的方法进行了比较。

  • TRAD：从源语料库中学到的分类模型直接在目标语料库中使用
  • $Baseline$：训练集测试集都来自同一语料库
  • DR：The dimensionality reduction based transfer learning approach
  • TCA：Transfer component analysis method
  • TSC：Transfer sparse coding approach
  • TNMF：Transfer non-negative matrix method
  • A-DAE：Adaptive denoising autoencoder

具体细节

情感分类器使用线性核SVM，在实验设置下，因为假设数据在目标语料库中未标记，使用交叉验证策略不可能自动地调整目标语料库的最佳参数，所以通过经验地搜索参数空间以获得最优参数来评估我们提出的方法，获得最优参数。

实验结果

  • 在所有的实验结果中都优于除了$Baseline$以外的方法，说明FLTSL具有鲁棒性。
  • 所有的迁移学习方法都明显优于TRAD，因为传统标准学习算法的一个主要局限性在于它将来自不同语料库的数据看作是从同一个语料库，但实际上在跨语料库识别问题上并不满足独立同分布。
  • FLTSL比当前最先进迁移学习算法表现都优秀，这些特征变换方法的主要局限在于忽略了特征选择策略或数据分布的内在几何特征。而FLSTSL能够同时进行迁移学习和子空间学习，并且加入图正则化来学习几何特征。
  • FSTSL优于基于尺寸缩减的转移学习算法，它们都没有考虑特征选择。

有效性证明

作者做了对比试验：

  • $\alpha$和$\gamma$设置为0，即不做特征选择。
  • $\gamma$设置为0，即不做图正则化。

结果是不做特征选择最差，不做图正则化次之，FLTSL最好，证明了有效性。

参数敏感性

FLTSL主要有四个参数$\alpha ,\beta ,\gamma,k$(最近邻个数)，作者对六个实验画出了分类正确率随参数变化的图。最优的参数分别是0.1，1，100，5.

结论

FLTSL可以学习跨语料库的特征，并且具有鲁棒性，比当前最先进的迁移学习表现要优秀。当前很多非监督和半监督方法都可以被纳入FLTSL框架。作者计划将特征选择策略引入到深度转移学习算法中，以进一步提高语音情感识别的识别性能。
